### 注释
单行
```python
# comment
```
多行
```
'''
comment1
'''

"""
comment2
"""
```
多行注释中最常使用双引号多行注释<br>

### 变量命名

大驼峰 `VarName`<br>
小驼峰 `varName`<br>
下划线 `var_name`<br>

### 数据类型
* 数值型 number
	* 整型 int
	* 浮点型 float
* 布尔型 boolean
	* 真 True
	* 假 False
* 字符串 str
* 列表 list
* 元组 tuple
* 集合 set
* 字典 dict

检查数据类型的函数: type()<br>
