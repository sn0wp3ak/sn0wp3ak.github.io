### 定义函数
```shell
function 函数名(){
   函数体
   return 返回值
}
```

* 定义函数可以用`function`声明, 也可以不用声明且不加任何参数的直接定义`函数名()`
* 函数的返回值, 可以用`return`返回, 如果不手动返回的话, 会以最后一条命令的执行结果作为返回值, 如果return后面是数字型返回值, 那么取值范围0-255

在函数的外部, 可以用`$?`来获取函数的返回值<br>

调用函数: 直接使用函数名, 不加`()`<br>

注意:
1. 函数在调用前必须先定义, 定义函数放在脚本开始
2. 调用函数仅使用函数名

### 函数的参数

Shell脚本中, 在调用函数时, 可以向函数传递参数, 并通过`$n`的形式来获取参数值<br>
