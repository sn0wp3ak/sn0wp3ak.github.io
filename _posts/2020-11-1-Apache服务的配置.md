配置文件的存放位置:<br>

| 配置文件         | 位置                       |
| :-:          | :-:                        |
| 服务目录     | /etc/httpd                 |
| 主配置文件   | /etc/httpd/conf/httpd.conf |
| 网站数据目录 | /var/www/html              |
| 访问日志     | /var/log/httpd/access_log  |
| 错误日志     | /var/log/httpd/error_log   |

主配置文件**httpd.conf**中有三种类型的信息:<br>

>1.注释<br>
>2.全局配置<br>
>3.区域配置<br>

全局配置: 作用于所有站点, 避免频繁写入重复参数<br>
区域配置: 针对于个别站点的单独配置的参数<br>

主配置文件常用参数:<br>

| 参数           | 用途                    |
| :-:            | :-:                     |
| ServerRoot     | 服务目录                |
| ServerAdmin    | 管理员邮箱              |
| User           | 运行服务的用户          |
| Group          | 运行服务的用户组        |
| ServerName     | 网站服务器的域名        |
| DocumentRoot   | 网站数据目录            |
| Directory      | 网站数据目录的权限      |
| Listen         | 监听的IP地址与端口号    |
| DirectoryIndex | 默认索引页页面          |
| ErrorLog       | 错误日志文件            |
| CustomLog      | 访问日志文件            |
| Timeout        | 网页超时时间, 默认300秒 |
