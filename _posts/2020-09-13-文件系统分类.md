Windows:
1. FAT  ( 文件分配表 File Allocation Table )
    * FAT12
    * FAT16
    * FAT32
2. exFAT   16EB/F ( 16EB = 16 * 1024 * 1024TB )
3. NTFS ( New Technology File System )

Linux:
1. ext
    * ext2
    * ext3
    * ext4

Apple:
1. HFS ( 分层文件系统 Hierarchical File System )
2. HFS+
3. APFS

Sun:
1. ZFS

`*/F    文件系统支持的最大文件大小`<br>
`*/V    文件系统支持的最大卷大小`
